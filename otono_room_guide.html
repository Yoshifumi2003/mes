操作方法

番号名称機能説明
①録音開始左上ボタンで新規トラックの録音を開始します。マイク使用許可を求められ、許可すると即座に録音が始まり、⑧のタイマーが動き、②が有効化されます。
②録音停止進行中の録音を終了し、トラックをタイムラインに配置します。
③ファイル追加ローカル PC などに保存された WAV／MP3 ファイルをトラックとしてインポートします。
④保存現在のプロジェクト（配置・設定など）を保存します。
⑤開く④で保存したプロジェクトファイルを読み込み、続きから編集できます。
⑥再生すべてのトラックをタイムライン先頭から再生します。
⑦一時停止再生を一時停止します。
⑧録音時間表示録音中のみ経過時間を秒単位で表示します。
⑨拍子設定3 拍子／4 拍子を切り替え、グリッド線と小節番号が自動で更新されます。
⑩テンポスライダーBPM（30–240）をリアルタイムで変更し、再生・録音速度を調整します。
⑪トラック名ラベル収録素材の名称を変更することができます。
⑫ボリュームスライダー各トラックの音量を個別に調整します。バーを左に移動で小さく、右に移動で大きくなります。
⑬トラッククリップクリップ（音素材）の実体です。ドラッグまたは長押しで横に移動することができます。また、クリックまたは長押しすると削除・複製ボタンがポップアップし、ワンタップで操作できます。
⑭トラックカット機能ドラッグまたは長押しでトラックを任意の長さにカットすることができます。

推奨環境

MediaRecorder API と File System Access API に対応した最新 Chrome／Edge に対応します。なお、Safari は録音機能が制限される場合があります。iOS 版については、順次公開予定です。
